# SCHEDULER

## サイト概要
お客様にアポイントの取りたい日時を投稿していただき、お客様自身はマイページで確認をすることでミスを防ぐことと手間の削減目的で用いていただく。
それに対し、管理者側はお客様の投稿が社内の予定表に反映されスケジュール管理ができるようになる。それに加えて他の営業マンの予定も確認ができるサイト。

### サイトテーマ
スケジュール調整管理サイト

### テーマを選んだ理由
前職で営業をしていてお客様とのアポイント調整に時間がかかり、業務の時間を圧迫していた経験から。

### ターゲットユーザ
企業の営業職

### 主な利用シーン
アポイントなどスケジュール調整
